PW-02

Protected Playwright runner

Browser execution runs in GitHub Actions for the v3001.123.0 board, not inside the GitHub Pages browser tab.

Implementation

  • Workflow: .github/workflows/run-playwright-job.yml.
  • Runner script: scripts/run-playwright-job.cjs.
  • Bridge entry: POST /playwright/jobs on the hosted Cloudflare worker.
  • Only scripts listed in script-registry.json can run.

Acceptance

The dashboard can create a job without exposing tokens, raw Playwright code, or arbitrary target URLs to the browser.
