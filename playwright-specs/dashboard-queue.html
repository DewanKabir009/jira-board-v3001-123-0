PW-03

Dashboard job queue

The 123 modern board includes a Playwright run console inside the spec playbook above Data Pull.

Implementation

  • Users choose a locked script, environment, and Jira ticket key.
  • The board posts a schema-valid job request to /playwright/jobs.
  • The bridge returns a durable job id, Actions URL, summary JSON URL, and evidence page URL.

Acceptance

The user can queue automation from the board and keep the job id visible without leaving the dashboard.
